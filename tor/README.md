#tor_dl
---

Example
```
tor_dl.py -url http://penbang.sysbase.org/other_projects/simple_xor.pdf -en ru,nz -xn us,ru
```

Arguments:

```
  -h, --help      show this help message and exit
  -p port         Socks Port (default: 7000)
  -url url        URL to download (default: None)
  -xn exit_node   Only Accepts Country Codes (default: us)
  -en entry_node  Only Accepts Country Codes (default: ru)
  -d destination  Download Destination
```
